# Feira do Gado

Marketplace web para negociação pecuária desenvolvido com HTML, CSS e JavaScript, aplicando conceitos de SDD (Spec Driven Development) e organização baseada em especificações.

## Sobre o projeto

O sistema simula uma plataforma moderna de compra e venda de gado bovino, permitindo que usuários visualizem animais disponíveis em diferentes estados do Brasil, realizem ofertas e simulem compras através de uma interface interativa e responsiva.

A aplicação possui foco em experiência visual, organização de componentes e separação de responsabilidades entre estrutura, estilo e comportamento.

# Exemplos de uso

## Explorar animais

O usuário pode acessar a seção de animais e visualizar:

- raça;
- peso;
- localização;
- valor;
- ofertas anteriores.

---

## Navegar pelo mapa

Ao selecionar um estado no mapa do Brasil, o sistema apresenta:

- quantidade de animais;
- vendedores ativos;
- média de preços;
- estatísticas regionais.

---

## Fazer oferta

O usuário pode abrir o modal de oferta e inserir um valor personalizado para negociação.

---

## Comprar animal

O sistema exibe um modal de confirmação simulando a conclusão da compra.
